# MN Daylight Lodge Website

## Overview
This is the official website for the **MN Daylight Lodge**, a part of the Minnesota Masons community. The website serves to showcase the lodge's history, provide information on upcoming events, and offer a member login section for exclusive content. It also includes a contact form for inquiries.

## Features
- **Homepage**: Introduction to MN Daylight Lodge and its mission.
- **Events Page**: Integrated Google Calendar to display upcoming lodge events.
- **History Page**: A section to learn about the rich history of the MN Masons.
- **Member Login**: A secure login page for members to access exclusive resources.
- **Contact Form**: A form for visitors to send inquiries or messages to the lodge.

## Technologies Used
- **HTML**: For the structure of the website.
- **CSS**: For styling the website to match the MN Masons theme.
- **Google Calendar Embed**: For displaying upcoming events.
- **Form Handling**: Placeholder for the contact form and login.
